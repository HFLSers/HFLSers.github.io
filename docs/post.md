# 投稿指南

该网站使用 Markdown 格式存储文章，[教程](https://markdown.com.cn/basic-syntax/)。

建议直接使用 Markdown 写作，或者使用 [Docx Markdown 转换器](https://word2md.com/) [^1]。

[^1]: [https://markdown.com.cn/basic-syntax/](https://markdown.com.cn/basic-syntax/) 和 [https://word2md.com/](https://word2md.com/) 与此网站无关。

## 分区投稿

我们目前将网站分为以下板块：

- **个人分享**: `personal/`
- **学术探究**: `study/`
- **社团活动**: `club/`
- **校园生活**: `life/`
- **校友风采**: `alumni/`

## 投稿步骤

### GitHub Issues

> 该方法适合不熟悉开发的同学。

1. 访问 [GitHub Issues](https://github.com/HFLSers/HFLSers.github.io/issues)。
2. 点击 "New issue" 按钮。
3. 选择 "New post" 模板。
4. 填写表单并提交。

### GitHub Pull Request

> 该方法适合熟悉 GitHub 的同学。

1. Fork 本仓库。
2. 将您的 Markdown文件放入上述对应目录，并更新 `zensical.toml`。
3. 提交 Pull Request。
